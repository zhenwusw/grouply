Grouply

## Intro

* Prototype, please see [here](http://pan.baidu.com/s/1hqy0jvI#dir/path=%2Fgrouply)

* `git clone http://github.com/zhenwusw/grouply`
* `bower install`
* Start server with `mix run --no-halt -e Grouply.start`
